# Hand Gesture Controlled Car

## Overview

The Hand Gesture Controlled Car is a wireless robotics project that uses an Arduino and MPU6050 motion sensor to control a robotic vehicle through hand movements. The system interprets gestures such as forward, backward, left, and right motions and transmits commands wirelessly to the car, enabling intuitive and touch-free control.

## Features

* Wireless gesture-based control
* Real-time motion sensing using MPU6050
* Forward, backward, left, and right movement control
* Arduino-based embedded system
* Low-cost and user-friendly design
* Interactive robotics application

## Components Used

* Arduino Uno/Nano
* MPU6050 Accelerometer and Gyroscope Sensor
* RF Module / Bluetooth Module
* Motor Driver (L298N)
* DC Motors
* Chassis and Wheels
* Battery Pack
* Jumper Wires

## Working Principle

The MPU6050 sensor mounted on a hand-held controller detects hand movements and orientation changes. An Arduino processes the sensor data and converts it into movement commands, which are transmitted wirelessly to the receiver module on the car. The receiving Arduino controls the motor driver to move the vehicle according to the detected gestures.
